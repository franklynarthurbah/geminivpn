║       GeminiVPN iOS App — Build & Distribution Instructions      ║
╚═══════════════════════════════════════════════════════════════════╝

iOS apps CANNOT be built on Linux. You need a Mac with Xcode.

═══ REQUIREMENTS ═══════════════════════════════════════════════════

• macOS 13+ (Ventura or later)
• Xcode 15+ (free from Mac App Store)
• Apple Developer Account ($99/year) — required to distribute
• CocoaPods: sudo gem install cocoapods

═══ OPTION A: Capacitor (Recommended — reuses the React frontend) ══

This wraps the existing React/Vite web app into a native iOS container.

1. On your Mac, clone the project:
   git clone <your-repo> geminivpn
   cd geminivpn/frontend

2. Install Capacitor:
   npm install @capacitor/core @capacitor/ios @capacitor/cli
   npm install @capacitor/haptics @capacitor/keyboard @capacitor/status-bar

3. Build the web app:
   npm run build

4. Add iOS platform:
   npx cap add ios
   npx cap sync ios

5. Open in Xcode:
   npx cap open ios

6. In Xcode:
   a. Set your Team (Apple Developer account) in Signing & Capabilities
   b. Set Bundle Identifier: com.geminivpn.app
   c. Set Version: 1.0.0 / Build: 1

7. Update capacitor.config.ts:
   server: { url: 'https://geminivpn.zapto.org', cleartext: false }

8. Archive & Export:
   Product → Archive → Distribute App → App Store Connect
   (or Ad Hoc for internal testing)

═══ OPTION B: React Native (Full native app) ═══════════════════════

1. Install React Native CLI:
   npm install -g @react-native-community/cli

2. Create project:
   npx react-native@latest init GeminiVPN --template react-native-template-typescript

3. Add WireGuard library:
   npm install react-native-wireguard
   cd ios && pod install

4. Port the React components from frontend/src/ to React Native
   (Screens: Login, Dashboard, ServerList, VPNConnect)

5. Archive same as Option A (Xcode → Archive)

═══ TESTFLIGHT DISTRIBUTION ════════════════════════════════════════

1. Upload IPA to App Store Connect (appstoreconnect.apple.com)
2. Add testers under TestFlight tab
3. Testers install via TestFlight app
4. No public App Store review needed for beta

═══ APP STORE SUBMISSION ═══════════════════════════════════════════

Required materials:
• Screenshots: 6.7" (iPhone 15 Pro Max), 12.9" (iPad Pro)
• App description, keywords, privacy policy URL
• Privacy policy: https://geminivpn.zapto.org/privacy

Review notes:
• VPN apps require Apple's Network Extension entitlement
• Submit a review note: "Uses WireGuard for encrypted VPN tunneling.
  Server at geminivpn.zapto.org"
• Expected review time: 1–3 business days

═══ ENTERPRISE DISTRIBUTION (No App Store) ══════════════════════════

With Apple Enterprise Program ($299/year):
• Sign IPA with enterprise cert
• Host at: https://geminivpn.zapto.org/downloads/GeminiVPN.ipa
• Users install via Safari with manifest.plist
